
### 1. 强制类型转换
- 变量名前加（类名）就是强制类型转换
- 从高到低的时候必须加强制类型转换
- 如果超出低的类型的范围，数据就会错误
```java
public static void main(String[] args){
    int i = 128;
    byte b = (byte)i;
    
}
```

### 2.自然类型转换
- 从低到高就是自动类型转换
- 如果是含计算的，那么自动转换是计算后再转换，比如
```java
int m = 10_0000_0000;
int y = 20;
//计算时还是int，已经溢出了，再转换还是错的
long total2 = m*y;
//先转换再计算，就能得出正确的值
long total3 = m*((long)y);
```
