## 位运算符
- &, |, ^（异或）, ~（取反）, >>（右移）, <<（左移）, >>>
- 

```java
public static void main(String[] args){
    /*
    A = 0011 1100
    B = 0000 1101
    
    A&B = 0000 1100 如果都是1，则为1，否则为0
    A|B = 0011 1101 其中1个为1，则为1，否则为0
    A^B = 0011 0001 两个不同则为1，否则为0
    ~B = 1111 0010 取相反
    
    0000 0000 0
    0000 0001 1
    0000 0010 2
    0000 0100 4
    0000 1000 8
    左移1位就是乘1次2
    右移1位就是除1次2
    */
    System.out.print(2<<3);//输出就是8
}
```
