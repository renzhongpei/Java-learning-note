1. 建立一个后缀为java的文件
2. 文件名和类名要一样,即Hello.java ----类名首字母用大写

```java
public class Hello{
    public static void main(String[] args){
        System.out.pring("Hello World!");
    }
}
```

3. 控制台到java的文件路径里运行javac Hello.java，就会发布class文件
4. 控制台运行java Hello，即能运行Java文件
